Kullanım Örnekleri

1. Basit Kullanım (HTML Attribute)

Sayfa yüklenirken veritabanınızdaki veriyi doğrudan JSON formatında basabilirsiniz. API yazmanıza gerek kalmaz.

<!-- Sadece bu HTML kodunu ekleyin -->
<div 
  data-pia-calendar="true" 
  data-pia-events='[
    {"title": "Toplantı", "date": "2024-10-15", "color": "primary"},
    {"title": "Sunum", "date": "2024-10-18", "color": "danger"}
  ]'>
</div>

<!-- JS Dosyasını Çağırın ve Başlatın -->
<script src="PiaCalendar.js"></script>
<script>
    PiaCalendar.observeDOM();
</script>
2. Gelişmiş Kullanım (REST API & AJAX)

Aylar arası gezerken veya veri eklerken kendi API'nize (PHP, .NET vb.) anlık istek atabilirsiniz.

<div data-pia-calendar="true"></div>

<script src="PiaCalendar.js"></script>
<script>
// MutationObserver ile sayfadaki tüm takvimleri izle ve API'ye bağla
PiaCalendar.observeDOM({
    onFetchEvents: async (year, month) => {
        const res = await fetch(`/api/takvim?y=${year}&m=${month}`);
        return await res.json(); 
    },
    onAddEvent: async (data) => {
        const res = await fetch('/api/ekle', {
            method: 'POST', body: JSON.stringify(data)
        });
        return await res.json(); // Eklenen veriyi id'si ile dönmeli
    },
    onDeleteEvent: async (id) => {
        await fetch(`/api/sil/${id}`, { method: 'DELETE' });
        return true;
    }
});
</script>
